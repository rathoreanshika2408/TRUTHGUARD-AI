TRUTHGUARD-AI
AI Hallucination Detection & Content Verification System

TRUTHGUARD-AI is an intelligent web-based platform designed to detect misleading, hallucinated, or potentially false AI-generated content. It helps users verify information reliability and promotes responsible AI usage.

📌 Repository: https://github.com/rathoreanshika2408/TRUTHGUARD-AI

📌 Project Overview
With the rapid rise of AI-generated content, distinguishing real information from fabricated responses has become challenging. TRUTHGUARD-AI aims to act as a digital truth assistant that evaluates text and flags suspicious or unreliable content.

This project includes:
A web interface for user interaction
A backend system for content analysis
A browser extension (planned/partial)
Community and login interface
Verification-focused AI logic

 Features
✔ AI-generated content verification
✔ Hallucination detection support
✔ Interactive web interface
✔ Login & community pages
✔ Modular backend using Python
✔ Lightweight and browser-friendly
✔ Extendable architecture for future models
✔ Web-based deployment support

🧠 How It Works
User enters text or content.
The backend processes the input.
AI-based logic analyzes reliability.
The system returns flagged insights.
Users review suspicious outputs.

⚙️ Installation Guide
Follow these steps to run the project locally.

Step 1 — Clone Repository
git clone https://github.com/rathoreanshika2408/TRUTHGUARD-AI.git

cd TRUTHGUARD-AI
Step 2 — Create Virtual Environment
python -m venv venv

Activate environment:

Windows
venv\Scripts\activate

Mac/Linux
source venv/bin/activate

Step 3 — Install Dependencies
pip install -r requirements.txt
Step 4 — Run Application
python app.py
Step 5 — Open in Browser
